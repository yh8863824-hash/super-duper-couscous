That's it! Three files = a full working website.

## STEP 5 — HOW TO SET UP YOUR FILES

### On Windows:
1. Right-click your Desktop → "New Folder" → Name it `would-you-rather`
2. Open the folder
3. Right-click inside → "New" → "Text Document"
4. Name it `index.html` (make sure to include `.html`)
5. Repeat for `style.css` and `script.js`
6. Right-click each file → "Open with" → Notepad (or VS Code)
7. Paste the code I gave you into each file
8. Save (Ctrl + S)

### On Mac:
1. Open TextEdit
2. Go to Format → "Make Plain Text"
3. Paste the code
4. Save As → choose your folder, name it `index.html`
5. Repeat for other files

### Best Free Code Editor:
Download **VS Code** (free): https://code.visualstudio.com
- Much better than Notepad
- Colors your code so it's easy to read
- Free forever

### To Preview Your Website:
1. Find `index.html` in your folder
2. Double-click it
3. It opens in your browser — that's your website!

## STEP 6 — PUTTING IT ONLINE (FREE HOSTING)

### Option 1: Netlify (BEST for beginners — completely free)
1. Go to https://netlify.com
2. Click "Sign Up" → use your email
3. Click "Add New Site" → "Deploy Manually"
4. Drag and drop your ENTIRE `would-you-rather` folder onto the page
5. Wait 10 seconds
6. 🎉 Your site is LIVE with a real URL!
7. You can set a custom domain later

### Option 2: GitHub Pages (also free)
1. Go to https://github.com and create an account
2. Click "New Repository"
3. Upload your 3 files
4. Go to Settings → Pages → Enable GitHub Pages
5. Your site goes live at: `yourusername.github.io/would-you-rather`

### Option 3: Vercel (free, very fast)
1. Go to https://vercel.com
2. Sign up, drag and drop your folder
3. Instant live URL

**Recommendation:** Use **Netlify** — it's the easiest for beginners.

## STEP 7 — HOW TO ADD GOOGLE ADSENSE

### Step-by-Step:
1. Go to https://adsense.google.com
2. Click "Get Started"
3. Enter your website URL
4. Fill in your payment info (so Google can pay you!)
5. Google will review your site (takes 1-3 days)
6. Once approved, you get a code snippet like this:

```html
<ins class="adsbygoogle"
     style="display:block"
     data-ad-client="ca-pub-1234567890123456"
     data-ad-slot="1234567890"
     data-ad-format="auto"
     data-full-width-responsive="true"></ins>
<script>(adsbygoogle = window.adsbygoogle || []).push({});</script>
```

7. Open your `index.html`
8. Find the comment: `<!-- PASTE YOUR GOOGLE ADSENSE CODE HERE -->`
9. Delete the placeholder `<div>` below it
10. Paste your AdSense code there
11. Re-upload to Netlify
12. Done! Ads will now show automatically

### Requirements for AdSense Approval:
- ✅ Real, useful content (your game is!)
- ✅ Privacy Policy page (add a simple one)
- ✅ At least 20-30 visitors before applying
- ✅ No copied content
- ✅ Site must be live on a real domain

## ADD NEW QUESTIONS (Super Easy!)

Open `script.js` and find the `ALL_QUESTIONS` array.
Add a new question in this exact format:

```javascript
{ 
  id: 99,               // ← Give it a unique number
  a: "Your option A",   // ← First choice
  b: "Your option B",   // ← Second choice
  cat: "funny",         // ← funny, gross, superpowers, money, food, or all
  funFact: "Interesting fact shown after voting" 
},
```

That's all it takes to add infinite content!
